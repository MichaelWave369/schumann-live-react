# EIRM Current Snapshot Report

The Current Snapshot Report lives inside the Local Watchtower panel.

It creates a claim-safe report from the current dashboard snapshot:

- Kp value
- solar-wind speed
- GOES X-ray class
- spectrogram image state
- feed state
- Watchtower states
- local Watchtower rules
- claim boundary text

## Actions

The report supports:

- **Copy Markdown** for sharing a readable status summary.
- **Export JSON** for saving a structured current-state receipt.

## Boundary

The report is a dashboard-awareness receipt only.

It is not:

- a diagnosis
- a prediction
- a personal-event explanation
- proof of causation
- a replacement for source data

Live/public feeds can be unavailable, delayed, blocked, or stale. The report only captures what the browser dashboard sees at the moment it is generated.
